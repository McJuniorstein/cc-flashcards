# Source format samples

PDFs in this directory are **format references**, not card sources. They are kept to evaluate extraction quality (how well [Marker](https://github.com/VikParuchuri/marker) handles NIST's typical layout) before downloading the canonical source set listed in [`../README.md`](../README.md).

**Cards must never cite a document from this directory.** Anything cited by a card lives in `/sources/` directly.

## Contents

| File | Why it's here |
|---|---|
| `NIST.SP.1308.pdf` | Early format-check download. Used to confirm NIST PDF layout is workable for the extraction pipeline. |
